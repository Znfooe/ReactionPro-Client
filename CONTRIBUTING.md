# 参与贡献

感谢你改进 ReactionPro Client。

## 提交前

1. Fork 本仓库并从最新默认分支创建功能分支。
2. 保持改动集中，不提交生成目录、日志、密钥或本机配置。
3. 新增资源时附上来源和可再分发许可证。
4. 在 `frontend/` 中运行：

```powershell
flutter pub get
flutter analyze
flutter test
```

## Pull Request

Pull Request 请说明：

- 用户可观察到的变化。
- 已运行的测试。
- 是否影响 Web、Windows 或移动平台。
- 是否引入新的依赖、字体、图片或其他第三方资源。

本仓库是公开镜像。通过审查的贡献会保留作者署名，回填到私有事实源完成集成验证，再同步回本仓库。

提交贡献即表示你有权按本仓库许可证提供该贡献。
